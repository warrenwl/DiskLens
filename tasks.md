# DiskLens Tasks

## 1. 说明

本文件用于把 `prd.md` 转换成可执行开发任务，服务于以下目标：

- 明确当前版本已经完成了什么
- 明确现在最值得继续推进的工作
- 让后续开发、测试、验收有共同清单

状态约定：

- `[x]` 已完成
- `[-]` 进行中 / 已有基础但仍需完善
- `[ ]` 未开始

优先级约定：

- `P0` 当前版本发布阻塞项
- `P1` 强烈建议在当前版本完成
- `P2` 可放入下一轮迭代

1. 先补 P0 自动化测试和异常验证。
2. 再做一次完整手动 QA，确保主流程闭环稳定。
3. 然后统一清理页文案、风险说明和权限提示。
4. 最后再进入分类规则增强、历史对比和体验优化。

## 9. 完成定义

以下条件同时满足时，可认为当前版本达到可发布基线：

- P0 任务全部完成
- 主流程手动 QA 通过
- `swift run DiskLensChecks` 通过
- `swift build` 通过
- `git diff --check` 通过
- 应用打包验证通过
- PRD、任务清单和当前实现不存在明显冲突
